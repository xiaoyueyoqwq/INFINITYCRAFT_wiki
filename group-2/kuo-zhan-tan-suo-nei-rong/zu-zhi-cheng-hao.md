# 👬 组织称号

如果你是一个组织的领导者，并且邀请了超过3个成员来到INFINITYCRAFT，你就可以向服务器管理申请组织注册，注册过后的组织可以申请组织称号，审核完毕后你的所有组织成员都将免费获得你所定制的组织称号并且在全服范围内展示

{% hint style="info" %}
<mark style="color:red;">**组织领导者必须先开通PRO会员后才可享受定制称号权益，成员无需重复开通**</mark>

称号颜色支持RGB渐变（BETA测试功能，可能出现问题）
{% endhint %}
