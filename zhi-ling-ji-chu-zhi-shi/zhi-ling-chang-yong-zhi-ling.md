# 🔧 指令 - 常用指令

| 指令              | 介绍                         |
| --------------- | -------------------------- |
| /rtp            | 随机传送一个安全地点                 |
| /back           | 返回到上一个死亡地点                 |
| /spawn          | 传送回服务器出生点                  |
| /w <玩家名>        | 向某位玩家私聊                    |
| /sit /lay /spin | /坐下/躺下/开转（？ni xian kai de） |
| /co i           | 查询方块信息（左右键均可用）             |
